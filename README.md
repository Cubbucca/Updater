Simple Folder Syncing Tool That will Launch The Copied App Once Cloned or If No Update is Needed
Will Only Copy Files Newer Than the Local Files
Will Copy appsettings (use at own risk)
If Published as Single File and Self Contained it Will Run without issue.
If not Published as Single File it will have conflict when trying to copy Microsoft.Extention dlls causing the app to most likey have a conflict with these remaining outdated
